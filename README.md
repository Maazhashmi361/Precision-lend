# PrecisionLend EMI Calculator

A professional, responsive loan payment calculator for computing EMI (Equated Monthly Installment), interest burden, and amortization schedules.

## Features

- 📊 Live EMI calculation with real-time updates
- 📈 Visual breakdown chart (principal vs interest)
- 📋 Detailed amortization schedule (all months)
- 🌙 Dark mode toggle
- 📥 Download schedule as CSV
- 📱 Fully responsive (mobile, tablet, desktop)
- 🔗 Shareable calculation URLs
- ⚡ Zero dependencies, pure HTML/CSS/JS
- ♿ Accessible (WCAG compliant)

## Usage

1. Download `emi-calculator-fixed.html`
2. Open in any modern browser
3. Enter loan amount, interest rate, tenure, and processing fee
4. View instant results and detailed amortization

## Deployment

### Cloudflare Pages (Recommended - Free)
1. Push this repo to GitHub
2. Connect to Cloudflare Pages
3. Deploy automatically on every push

### Local
Simply open the HTML file in your browser—no server needed.

## Tech Stack

- HTML5
- CSS3 (custom design system)
- Vanilla JavaScript (no frameworks)
